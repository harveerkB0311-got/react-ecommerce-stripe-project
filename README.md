# 🛒 React E-Commerce with Stripe Payments

A full-stack e-commerce web application built with React, Node.js, MySQL, and Stripe.  
This project demonstrates real-world development skills including frontend UI, backend APIs, database design, and payment integration.

---

## 🚀 Overview

This application allows users to browse products, add items to a cart, and securely complete payments using Stripe.

Built to showcase:
- Full-stack development
- REST API integration
- Payment gateway integration
- Scalable application structure
- Real-world business logic

---

## ✨ Features

- 🛍️ Product listing with dynamic data  
- 🛒 Shopping cart functionality  
- 💳 Secure checkout using Stripe  
- 🔗 REST API integration (Node.js backend)  
- 📦 MySQL database schema  
- 📱 Responsive UI design  
- ⚡ Clean and maintainable code  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js  
- JavaScript (ES6+)  
- CSS3  
- Axios  

**Backend:**
- Node.js  
- Express.js  
- MySQL  

**Other:**
- Stripe API  
- REST APIs
